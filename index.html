Morrowind Key Mapper

Morrowind Key Mapper is a DInput wrapper that remaps key codes to enable the use of non-US keyboards in Morrowind.

Details

See the README on Git Hub for more details. link

Installing

Currently, if you want to use the Morrowind Key Mapper you need to download the code, build the Visual Studio projects, and copy the DLLs into your Morrowind folder under Program Files.
